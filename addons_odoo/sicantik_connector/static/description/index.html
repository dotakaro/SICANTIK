üöÄ Quick Start

  1. Install the module from Apps menu
  2. Go to SICANTIK > Configuration > API Configuration
  3. Click "Test Connection" to verify API access
  4. Click "Sync Now" to fetch initial data
  5. View permits in SICANTIK > Permits menu
