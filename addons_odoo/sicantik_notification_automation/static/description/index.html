SICANTIK Notification Automation

Sistem Notifikasi Massal dan Otomatis

Modul ini menyediakan sistem notifikasi massal dan otomatis berdasarkan pemicu/event tertentu. Dengan modul ini, Anda dapat mengkonfigurasi berbagai skenario notifikasi tanpa perlu modifikasi kode.

Fitur Utama

Notifikasi Berbasis Aturan

Sistem aturan notifikasi yang dapat dikonfigurasi melalui UI

Pemicu Berbasis Event

Pemicu berdasarkan event Odoo (create, write, state change)

Kampanye Massal

Kampanye notifikasi massal dengan pemrosesan batch

Manajemen Antrian

Antrian pengiriman dengan mekanisme ulang coba

Cron Jobs

  • SICANTIK: Process Notification Queue - Setiap 1 menit (Priority 20)
  • SICANTIK: Process Campaign Batch - Setiap 5 menit (Priority 15)
  • SICANTIK: Cleanup Old Notification Logs - Setiap hari (Priority 5)

Semua cron jobs dapat dimonitor melalui: Settings → Technical → Automation → Scheduled Actions
